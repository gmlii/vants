

#object#示例目标

* 目标-添加英文版介绍，新闻：
 - 添加enabt:英文版介绍
 - 添加enews:英文版新闻
 - 添加相关新闻评论

#addmodel#添加(复制)模型

* 复制[文档/资讯]等模型：
 - 架设 > 根模块架设 > 文档/资讯 :
 - 用about复制出enabt:英文版介绍; 
 - 用news复制出enews:英文版新闻; 
 - 其他：[互动/评论]等附属模型也复制一份
 - 当然可添加...不过麻烦点。

#addfields#设置字段栏目

- [文档/资讯]设置入口：架设 > 根模块架设 > 文档/资讯 > [栏目/字段]
- [互动/评论]设置入口：架设 > 根模块架设 > 互动/评论 > [字段]

#addadmin#添加管理入口

* 设置入口：架设 > 菜单导航配置 > 
* 规划：
 - 单独设置一个顶级目录
 - 也可放在现有的顶级目录下
* 规范：规范如下，可参考现有配置

```
// 一行只有一个连接：直接填连接
?file=admin/groups
// 一行多个连接:格式1:直接填html代码：
<a href="?file=admin/catalog" title="文档栏目">栏目管理</a> - 
<a href="?file=admin/catalog&mod=adpic" title="广告栏目">广告</a>
// 一行多个连接:格式2：
站点介绍(!)?file=dops/a&mod=about
增加(!)about(!)jsadd
// 一行多个连接:格式3：
新闻评论(!)?file=dops/a&mod=nrem(!)
栏目(!)?file=admin/catalog&mod=news(!)frame
// 一行多个连接:格式3：
产品分类(!)?file=admin/catalog&mod=cargo(!)frame
设置(!)?file=apis/exp_order

```
