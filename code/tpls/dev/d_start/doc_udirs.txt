

#ueadm#修改后台入口

* 目的：为安全和自定义特色！

* 移动文件：
 - /root/下创建usecret目录（自定义），
 - 复制/root/run/adm.php到此目录，并改名cat.php（自定义）
 - 修改文件中_paths.php路径：`require(dirname(dirname(__FILE__)).'/run/_paths.php');`

* 修改配置：
 - 文件：/code/cfgs/excfg/ex_vopfmt.php
 - 修改项：$_ex_vopfmt['tpl']，
 - 修改为：`'adm' => '/usecret/cat.php',`//目录和路径与上述修改的一致

* 访问Enjoy！：
 - /root/usecret/cat.php
