#func#功能模块

* 数据 : 分享/同步/外部导入/采集/推送/旧版导入
* 接口 : 微信/短信/ip地址/地图(baidu,google)/支付(paypal,支付宝,财付通)
* 其它 : 计划任务/积分/模板继承/tag缓存/js标签/
* 文档/资讯(默认) : 商品展销/新闻/专题/课程资源/样例文档/内部公文/问答系统
* 用户/会员(默认) : 企业/政府机构/非盈利组织/个人/管理员
* 展示版本(默认) : 电脑版/手机版/开发演示版/Guides(英文)/管理后台/会员中心
