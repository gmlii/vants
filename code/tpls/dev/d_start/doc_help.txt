#lang#翻译说明

* 文件架构
```
  [code]
	├─lang
	│  ├─dbins 数据库架构数据 语言包
	│  ├─kvphp 核心脚本 语言包
	│  ├─ptinc 文本块 语言包
	│  └─ucfgs 配置数组 语言包
```

* 多语言版本
 - 后缀说明：-cn:中文; -en:英文; -fr:法语; -ru:俄语; -es:西班牙语; -de:德语; -jp:日语; -kr:韩语; -ar:阿拉伯语
 - 中文为默认语言，英文语言包还需要完善；
 - 后续批次，愿有志愿者翻译 法语,俄语,西班牙语的语言包；
 - 其次就是：德语,日语语言包；
 - 再次就是：韩语,阿拉伯语；
 - 当然愿意：有志愿者翻译更多其他语言包！
 - 欢迎参入，感谢奉献！

* 增加语言包做法
 - 假设增加[-fr:法语]语言包：
 - 吧如上目录的[-cn]所有文件，复制一份并改名为[-fr]后缀，然后翻译即可
 - 可用机器翻译后，人工修改修正一下；
 - 如果增加某种语言，且可由现有语言转换而得，可复制用转换工具转换而得，如增加繁体中文，可由简体中文用工具转化即可
