

#prboot#boot参数

* 目录：/code/cfgs/boot/
* const.cfg.php : Cookie前缀,自定义配置,调试日志配置
* cfg_load.php : 类自动加载规范，二次开发可能需要配置
* cfg_db.php : 数据库配置，更换数据库请配置
* cfg_adbug.php : 辅助调试工具的账号密码
  -> 起始页：/root/index.php?start 有些链接为常用工具，需要这里设置 或 后台登录才可使用


#prexcfg#excfg参数

* 目录：/code/cfgs/excfg/
* ex_mail.php : 邮件参数
* ex_sms.php : 短信
* ex_vopfmt.php : 展示模版
* ex_outdb.php : 数据分享同步，外部数据库 配置参数
* ex_a3rd.php : 第三方[支付/登录]配置


#prsycfg#sycfg参数

* 目录：/code/cfgs/sycfg/
* sy_keepid.php 系统保留ID配置
* sy_uachk.php 搜索引擎，浏览器，UserAgent关键字配置
* 系统参数，根据需要可修改


#prelse#其他参数

* 微信配置：后台 >> 插件 >> 微信接口 (支持多功能公众号)
