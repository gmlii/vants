#up31#更新指引

* 方式1. 安装新系统，导入旧数据 (建议)
 - 先安装新系统，再导入旧数据；

* 方式2. 自带升级程序升级
 - 先手动更新：\code\core\sdev\下 updAdmin.php, updBase.php 两文件；
 - 先手动更新：/root/tools/setup下文件 (除index.php安装入口)；

 - 安装后 更新缓存：后台 >> 工具 >> 系统工具 >> 系统缓存 >> 系统缓存：更新清理 >> 提交 
 - 通用说明见：<a href="?uplog">通用更新说明</a> :
