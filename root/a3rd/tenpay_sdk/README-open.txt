                                               腾讯财付通开放平台PHP-SDK


什么是财付通开放平台
--------------------------
财付通开放平台是中国领先的第三方在线支付平台财付通全新打造的一个应用平台，通过这个平台，第三方开发商的应用可以进入QQ钱包等渠道，
被1亿财付通用户、甚至10亿QQ用户所使用。
财付通之前已经为广大网民提供了很多生活应用，并且受到广泛欢迎，现在为了给更多用户提供更多生活、娱乐方面的便利，财付通现在对外开
放自己的应用平台，只要您是一家有能力的、正规、诚信经营的公司，可以开发出有创意的电子商务应用，就可以将您的应用嵌入到财付通开放
平台中。

详情请登录：http://open.tenpay.com

内容
--------

本压缩包包含:

  /classes                      -运行本sdk需要的处理类。
  /examples                     -SDK使用示例，可以访问index.php进行支付演示


基于
------------
- PHP 5



开发应用流程
----------------
1.注册成为开发者:在open.tenpay.com通过财付通帐号注册。
2.注册开发应用：在open.tenpay.com里申请。
3.登陆open.tenpay.com，获取开发者的商户号、key、及测试帐号密码、余额等信息。
4.登陆open.tenpay.com获取sdk及沙箱测试工具。
5.通过沙箱测试工具切换至沙箱测试环境，登陆QQ客户端，并进入qq小钱包进行测试。
6.登陆open.tenpay.com，申请应用上架。
